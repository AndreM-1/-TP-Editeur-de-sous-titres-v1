Pré-requis avant de lancer l'application :

- Pour la base de données, j'ai utilisé PostgreSQL. Il faudrait donc créer une base de données sous PostgreSQL qui se nomme "JavaEE_Editeur_sous_titres" en localhost avec Port=5432, login=postgres et mot de passe=admin
- Il faudrait créer sous la racine le répertoire : "C:\fichiersTempEditeurSousTitres" pour stocker les fichiers temporaires lors de l'upload des fichiers de sous-titres.
- Une fois l'application lancée sous Eclipse, il faut copier l'URL sur un vrai navigateur pour que l'affichage soit correct, car sur le navigateur d'Eclipse, cela ne sera pas le cas.
